ul vs. ol

  • unordered list
    • Html Tag ul
    • listet Punkte mit einem Punkt auf
    • Beispiele
      • To Do Liste
      • Einkaufsliste
      • Ikea Möbel Liste
    • Weiterführende Links
  • ordered list
    • Html Tag ol
    • listet Punkte mit einer Zahl auf
    • Beispiele
      1. Inhaltsverzeichnis
      2. Prozessbeschreibung
      3. Anfahrtsbeschreibung Google Maps
    • Weiterführende Links
